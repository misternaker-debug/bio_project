# Предсказание тканеспецифичной экспрессии генов по последовательности ДНК

Данный проект посвящён разработке модели глубокого обучения, которая по последовательности ДНК в окрестности старта транскрипции (TSS) гена предсказывает уровень его экспрессии в различных тканях человека. Используются данные проекта GTEx (v8) и аннотация GENCODE v39.

## Описание проекта
**Цель:** предсказать количественный уровень экспрессии гена (log2(TPM+1)) для 54 тканей, используя только последовательность ДНК длиной 10 kb вокруг TSS (±5 kb).  
**Зачем это нужно:** понимание регуляторного кода, интерпретация некодирующих мутаций, связанных с заболеваниями.

**Основные этапы:**
- Сбор и предобработка данных (GTEx, GENCODE, референсный геном hg38).
- Разведочный анализ (EDA): распределение экспрессии, корреляция тканей, PCA, связь GC-состава и экспрессии.
- Построение свёрточной нейронной сети (CNN) с residual-блоками для многозадачного предсказания.
- Оценка качества (Pearson R, RMSE) и сравнение с базовыми моделями.
